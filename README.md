# Tic-Tac-Toe

This is a simple game of Tic-Tac-Toe built using HTML, CSS and JavaScript.

## How to Play

The game is played between two players. Each player takes turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Features

* Designed using HTML, CSS and JavaScript
* Two Player Game
* Ability to Reset the game
* The game ends when one of the players has completed a row, column or diagonal
* Automatically switches turns between players
